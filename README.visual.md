## 🗂️ Project Structure

```text
manga-portfolio-website/
├── app/            # Next.js App Router pages
├── components/     # UI & manga-inspired components
├── lib/            # Utilities, data, hooks
├── styles/         # Tailwind & global styles
├── types/          # TypeScript types
└── public/         # Static assets
```

## 🎨 Design Philosophy

- **Panel-based layouts**: Manga-style content panels
- **Sequential reveal**: Animations as you scroll
- **Monochrome palette**: Black, white, grayscale
- **Manga typography**: Bold headers, clean text
- **Visual FX**: Halftone, ink, speech bubbles

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the app.

## 🛠️ Features

- Next.js App Router & Server Components
- TypeScript strict mode
- Tailwind CSS v4 custom theme
- Framer Motion page & scroll animations
- Lenis smooth scrolling
- shadcn/ui (planned)
- Responsive, mobile-first design
- Manga-inspired UI/UX
